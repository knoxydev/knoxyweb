Apps

A list of applications that I use and can recommend to others.

  • Sublime Text - Perhaps the best text & code editor.
  • Fluent Terminal / GitHub - A Terminal Emulator based on UWP and web technologies.
  • Micro / GitHub - A simple terminal-based text editor.
  • Alacritty / GitHub - A cross-platform, OpenGL terminal emulator. My current terminal.
  • bat - Print the code in the terminal.
  • fx / GitHub - Terminal JSON viewer & processor.
  • fd - A simple, fast and user-friendly alternative to 'find'.
